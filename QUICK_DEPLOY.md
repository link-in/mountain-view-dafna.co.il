# הוראות העלאה מהירה לשרת PHP

## שלב 1: בניית הפרויקט

```bash
npm run build
```

לאחר הבנייה, תתקבל תיקייה `out` עם כל הקבצים הסטטיים.

## שלב 2: העלאת קבצים לשרת

העלה את כל התוכן מתיקיית `out` לשרת שלך (בדרך כלל לתיקייה `public_html` או `www`).

### קבצים חשובים להעלאה:

1. **כל התוכן מתיקיית `out`** → העלה לשורש האתר
2. **קובץ `.htaccess`** → העלה לשורש האתר (כבר מוכן בפרויקט)
3. **קובץ PHP**: `public/api/ical.php` → העלה ל-`/api/ical.php` בשרת

## שלב 3: מבנה התיקיות בשרת

```
/public_html (או www)
├── index.html
├── .htaccess
├── _next/
│   └── static/
├── api/
│   └── ical.php
└── [כל שאר הקבצים מ-out]
```
