## ✅ בנייה הושלמה בהצלחה!

הפרויקט נבנה בהצלחה! כל הקבצים מוכנים בתיקיית `out`.

## 📦 מה להעלות לשרת

### העלה את **כל התוכן** מתיקיית `out` לשרת:

```
out/
├── index.html              ← העלה
├── .htaccess              ← העלה (אם לא קיים)
├── _next/                 ← העלה את כל התיקייה
│   └── static/            ← כל הקבצים הסטטיים
├── about/                 ← העלה
├── blog/                  ← העלה
├── contact/               ← העלה
├── home/                  ← העלה
├── portfolio/             ← העלה
├── service/               ← העלה
├── shop/                  ← העלה
├── photos/                ← העלה (כולל logo.png)
│   ├── logo.png          ← הלוגו החדש
│   └── [תמונות אחרות]
└── [כל שאר הקבצים]       ← העלה הכל
```

### קבצים נוספים:

1. **`.htaccess`** (מהשורש של הפרויקט) → העלה לשורש האתר
2. **`public/api/ical.php`** → העלה ל-`/api/ical.php` בשרת

## 🚀 איך להעלות

### דרך FTP/FileZilla:

1. התחבר לשרת דרך FTP
2. עבור לתיקייה `public_html` (או `www`)
3. **העלה את כל התוכן מתיקיית `out`** (אפשר להעלות zip ולחלץ)
4. העלה את `.htaccess` לשורש
5. צור תיקייה `api` והעלה את `public/api/ical.php` לתוכה

### דרך cPanel File Manager:

1. פתח את File Manager
2. עבור ל-`public_html`
3. העלה את כל הקבצים מתיקיית `out` (אפשר להעלות zip ולחלץ)
4. העלה את `.htaccess` לשורש
5. צור תיקייה `api` והעלה את `ical.php` לתוכה

## 🔧 פתרון בעיות

### בעיה: דפים לא נטענים
- ודא ש-`.htaccess` מועלה ופועל
- בדוק שהשרת תומך ב-`mod_rewrite`
- בדוק הרשאות קבצים

### בעיה: תמונות לא נטענות
- ודא שכל התיקיות `_next/static` מועלות
- בדוק שהלוגו `photos/logo.png` קיים
- בדוק הרשאות (755 לתיקיות, 644 לקבצים)

### בעיה: PHP לא עובד
- ודא שהשרת תומך ב-PHP 7.4+
- בדוק שהקובץ `ical.php` בנתיב הנכון: `/api/ical.php`
- בדוק הרשאות (644)

### בעיה: לוח שנה לא עובד
- בדוק שהקובץ PHP עובד: `https://mountain-view-dafna.co.il/api/ical.php`
- פתח את הקונסול בדפדפן (F12) ובדוק שגיאות
- ודא שה-URL ב-`CalendarView.tsx` מצביע ל-`/api/ical.php`

### בעיה: פופאפ הזמנות לא עובד
- פתח את הקונסול בדפדפן (F12) ובדוק שגיאות
- ודא ש-jQuery ו-jQueryUI נטענים
- בדוק שה-widget ID נכון: `bookWidget-157320-306559-0-1767278632`
