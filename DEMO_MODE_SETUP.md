## 🎯 איך זה עובד?

### Logic Flow

```typescript
1. User logs in as demo@hostly.co.il
   ↓
2. Session gets user.isDemo = true
   ↓
3. getDashboardProvider(user) detects isDemo
   ↓
4. Returns mockDashboardProvider instead of Beds24
   ↓
5. Dashboard shows mock data + Demo banner
```

### קבצים שונו

#### Backend:
- `src/lib/auth/types.ts` - הוסף `isDemo` ל-User & AuthUser
- `src/lib/auth/getUsersDb.ts` - מיפוי `is_demo` מה-DB
- `src/lib/auth/authOptions.ts` - העברת `isDemo` ל-JWT & session
- `src/lib/dashboard/getDashboardProvider.ts` - זיהוי demo user

#### Frontend:
- `src/app/dashboard/DashboardClient.tsx` - באנר Demo Mode
- `src/lib/dashboard/providers/mock.ts` - 40 הזמנות + שנת מחירים

#### Database:
- `supabase-migrations/006_create_demo_user.sql` - טבלה + demo user
