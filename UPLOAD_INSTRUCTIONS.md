# הוראות העלאה לשרת - mountain-view-dafna.co.il

## ✅ שלב 1: בנייה הושלמה

הפרויקט נבנה בהצלחה! תיקיית `out` מכילה את כל הקבצים הסטטיים.

## 📤 שלב 2: מה להעלות לשרת

### העלה את **כל התוכן** מתיקיית `out` לשרת שלך:

```
out/
├── index.html          ← העלה
├── _next/              ← העלה את כל התיקייה
├── about/              ← העלה את כל התיקייה
├── blog/               ← העלה את כל התיקייה
├── contact/            ← העלה את כל התיקייה
├── home/               ← העלה את כל התיקייה
├── portfolio/          ← העלה את כל התיקייה
├── service/            ← העלה את כל התיקייה
├── shop/               ← העלה את כל התיקייה
└── [כל שאר הקבצים]    ← העלה הכל
```

### קבצים נוספים שצריך להעלות:

1. **`.htaccess`** (מהשורש של הפרויקט) → העלה לשורש האתר
2. **`public/api/ical.php`** → העלה ל-`/api/ical.php` בשרת

## 📁 מבנה התיקיות בשרת (לאחר העלאה)

```
/public_html (או www)

└── [כל שאר הקבצים מ-out]
```

## 🚀 איך להעלות

### דרך FTP/FileZilla:
1. התחבר לשרת דרך FTP
2. עבור לתיקייה `public_html` (או `www`)
3. העלה את **כל התוכן** מתיקיית `out`
4. העלה את `.htaccess` לשורש
5. צור תיקייה `api` והעלה את `ical.php` לתוכה

### דרך cPanel File Manager:
1. פתח את File Manager
2. עבור ל-`public_html`
3. העלה את כל הקבצים מתיקיית `out` (אפשר להעלות zip ולחלץ)
4. העלה את `.htaccess`
5. צור תיקייה `api` והעלה את `ical.php`
