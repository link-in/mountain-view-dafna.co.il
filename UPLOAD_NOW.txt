===============================================================
  HOW TO UPLOAD
===============================================================

Option 1: FTP (FileZilla/WinSCP)
---------------------------------
1. Connect to your server via FTP
2. Navigate to public_html
3. Upload ALL contents from the "out" folder
4. Make sure .htaccess is uploaded

Option 2: cPanel File Manager
------------------------------
1. Open cPanel
2. Click File Manager
3. Navigate to public_html
4. Upload all files from "out" folder
5. Or upload as ZIP and extract

Option 3: Quick ZIP Upload
---------------------------
1. Compress the "out" folder to ZIP
2. Upload ZIP to server (via cPanel)
3. Extract ZIP in public_html
4. Delete ZIP file
